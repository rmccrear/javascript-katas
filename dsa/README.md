# DSA Katas

Focused data structures and algorithms practice in JavaScript.

## Kata List

1. **[Kata 1: Print Forward Without Spaces](./kata-01-print-forward-no-spaces.md)** - Loop through a string from front to back
2. **[Kata 2: Print Backward](./kata-02-print-backward.md)** - Loop through a string from back to front
3. **[Kata 3: Count Letter A](./kata-03-count-letter-a.md)** - Count matching characters in a string
4. **[Kata 4: Contains At Least Three A's](./kata-04-contains-three-as.md)** - Turn a counter into a true/false result
5. **[Kata 5: Palindrome Checker](./kata-05-palindrome.md)** - Strings, pointers, normalization, and edge cases

## How to Use These Katas

1. Read the problem statement.
2. Write the function from scratch.
3. Test the examples manually or with a small test file.
4. Refactor after the solution works.
5. Try the stretch goals if the core problem feels easy.
