# ES6 Katas

Focused practice with fluent string and array method chaining in JavaScript.

## Kata List

1. **[Kata 1: Trim and Lowercase](./kata-01-trim-lowercase.md)** - Chain simple string cleanup methods
2. **[Kata 2: Remove Spaces](./kata-02-remove-spaces.md)** - Split and join a string
3. **[Kata 3: Reverse a String](./kata-03-reverse-string.md)** - Convert a string to an array and back
4. **[Kata 4: Count Letter A](./kata-04-count-letter-a.md)** - Use `filter` and `length`
5. **[Kata 5: Contains At Least Three A's](./kata-05-contains-three-as.md)** - Turn a filtered count into a boolean
6. **[Kata 6: Clean and Reverse a Phrase](./kata-06-clean-and-reverse-phrase.md)** - Combine cleanup and reverse steps
7. **[Kata 7: Palindrome Chain](./kata-07-palindrome-chain.md)** - Build a palindrome checker with method chaining

## How to Use These Katas

1. Read the examples.
2. Write the function with a short chain of string or array methods.
3. Run the test cases.
4. Refactor the chain until it reads clearly from left to right.
